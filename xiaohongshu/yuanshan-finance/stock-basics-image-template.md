# 远山财经｜股市入门图片组模板规范

> 适用范围：小红书「股市入门」系列图片组。  
> 栏目定位：每期只讲一个股市知识单元，用简洁、系统、可连续追更的方式，帮助小白一块一块拼好股市认知。

---

## 1. 栏目基础设定

### 栏目名称

固定使用：

```text
股市入门｜第X期
```

不要使用：

```text
股市入门100讲
小白股市100讲
XX系列第X讲
```

原因：不预设总期数，避免给内容生产造成压力，也便于后续无限延展。

### 每期主题

每期只讲一个「知识单元」，而不是拆成过碎的词条。

推荐：

```text
股票
股价
市值
指数
板块
成交量
K线
支撑位
压力位
均线
```

不推荐把过于基础或高度相关的概念拆成多期，例如：

```text
高开、低开、平开分别单独一期开讲
5日线、10日线、20日线分别单独一期开讲
```

应合并成：

```text
开盘信号
短中长期均线
```

## 2. 封面模板规范【已固化】

### 封面比例

```text
3:4 竖版
```

### 封面整体气质

关键词：

```text
克制 / 简洁 / 知识感 / 高级 / 留白 / 系列化
```

封面不是信息页，只负责让读者一眼看清：

1. 这是远山财经的内容
2. 这是股市入门系列
3. 这是第几期
4. 本期讲什么概念

### 封面固定结构

从上到下依次为：

```text
1. 顶部居中：远山财经 LOGO
2. LOGO 下方：股市入门｜第X期
3. 中上部偏中位置：本期概念大标题
4. 标题下方：一句问题式副标题
5. 底部：极简装饰线 / 小型上涨折线图标
```

### 封面标题位置

大标题位置固定在页面高度的：

```text
约 1/3 到 2/5 区间
```

注意：

- 不要把大标题放得过低
- 不要让 LOGO 和期数占据过多空间
- 大标题是封面视觉核心
- 其他元素都为大标题服务

### 封面标题字号

当前已确认：

```text
大标题字号保持很大，视觉上必须强势
```

后续即使概念名称变长，也优先通过以下方式解决：

1. 精简概念命名
2. 合理分两行
3. 减少其他元素占位

不要轻易把大标题字号缩小。

### 封面元素限制

默认不放人物。

原因：

- 给标题留足空间
- 方便长概念名排版
- 保持封面系列感
- 防止画面变花

封面也不放：

```text
复杂插图
证书插画
过多图标
下期预告
页码
过多解释文字
```

### 封面 LOGO 规则

必须严格使用「远山财经」官方 LOGO：

```text
山形图标
中文：远山财经
英文：YUANSHAN FINANCE
```

禁止：

```text
擅自改 LOGO 造型
改英文拼写
生成替代山形图标
用圆形 logo 替代
用纯文字替代完整 logo
```

### 封面期数标签

固定文案：

```text
股市入门｜第X期
```

视觉要求：

- 不要太大
- 不要抢标题
- 但要足够清晰
- 对比度要明显
- 在小红书信息流中一眼能看清

推荐形式：

```text
暖红 / 赭橙色描边胶囊标签
或低饱和暖红底色 + 浅色文字
```

## 3. 色彩规范

### 主背景

```text
米白 / 暖米色 / 纸感背景
```

推荐气质：

```text
温和、干净、耐看、有知识感
```

### 主文字

```text
深棕黑 / 炭黑 / 深咖色
```

### 主题强调色

不用绿色作为主题色。

原因：中国股市语境下，绿色容易联想到下跌。

优先使用接近红色的暖色系：

```text
赭橙色
陶土红
砖红
豆沙红
暖棕红
低饱和橘红
```

使用原则：

- 用于期数标签、重点词、分割线、小图标
- 不大面积铺满
- 不使用过亮、过艳、刺眼的红色

### 辅助色

可少量使用：

```text
浅卡其
灰棕
浅米灰
深棕线条
```

如人物衣服需要区分主题色，可使用：

```text
低饱和蓝色 / 藏蓝 / 蓝灰色
```

## 4. 人物 IP 规范

### 固定人物

人物 IP 名称：

```text
远山学长
```

### 人物形象特征

```text
年轻男性
圆框眼镜
短发偏蓬松
温和表情
知识型 / 老师感 / 陪伴感
```

### 人物穿搭

已调整方向：

```text
白色衬衫
低饱和蓝色 / 蓝灰色开衫
卡其色长裤
棕色鞋子
```

不要继续使用绿色开衫作为主形象服装。

### 人物使用位置

默认不出现在封面。

优先出现在：

```text
内容页右上角 / 右侧辅助讲解位
尾页引导关注区域
解释复杂概念时的小老师位置
```

人物必须是辅助，不抢正文和标题。

## 5. 分页内容模板规范

### 分页风格

分页可比封面信息量更高，但仍要保持：

```text
清晰 / 克制 / 有逻辑 / 不花哨
```

适合使用：

```text
三段式模块
编号圆点
浅色卡片
小图标
简单示意图
重点词暖红强调
```

不推荐：

```text
堆满大段文字
过多装饰
过多插图
每页塞太多知识点
```

### 推荐分页结构

多数概念建议 5-6 页即可，不强行做 7 页。

基础结构：

```text
第 1 页：封面
第 2 页：这个概念是什么
第 3 页：为什么会有 / 为什么重要
第 4 页：它包含什么 / 买到什么 / 怎么理解
第 5 页：背后逻辑 / 影响因素 / 与其他概念的关系
第 6 页：一句话总结 + 关注引导
```

不固定要求每期都有「误区页」。

只有当误区足够有认知含量时，才单独加误区页。不要为了凑页数硬写误区。

### 尾页规范

尾页不写下期预告。

原因：避免给创作者造成固定更新压力，也避免后续选题顺序被绑定。

尾页只做：

```text
本期总结
核心结论
关注 / 收藏引导
```

推荐文案方向：

```text
关注远山财经，慢慢拼好股市认知。
```

或：

```text
点赞收藏，和远山财经一起打好金融基础。
```

## 6. 内容标准化流程【生图前必须执行】

以后每一期不要直接生图，必须先走内容流程。

### Step 1：确定本期知识单元

例：

```text
第 1 期：股票
第 2 期：股价
第 3 期：市值
```

### Step 2：检索权威资料

优先参考：

```text
中国投资者网
中国证监会 / 证监局投教资料
上交所 / 深交所投教资料
证券业协会 / 基金业协会资料
权威教材或监管口径资料
必要时补充 SEC Investor.gov 等海外投教资料
```

### Step 3：提炼核心逻辑

不要只是复制定义，要拆出：

```text
它是什么
它为什么存在
它解决了什么问题
普通投资者应该怎么理解
它和后续概念有什么关系
```

### Step 4：形成图文大纲

先输出文字大纲，让用户确认。

不要直接生成图片。

### Step 5：形成逐页定稿文案

每页必须包含：

```text
页面标题
正文
重点词
配图建议
```

### Step 6：用户确认后再生图

生图时严格遵守：

```text
封面模板
分页模板
LOGO 规范
配色规范
人物规范
不使用绿色主题色
不写下期预告
```

## 7. 第 1 期《股票》内容结构示例

### 第 1 页｜封面

```text
股市入门｜第1期

股票

买股票，到底买的是什么？
```

### 第 2 页｜股票是什么？

核心结构：

```text
01 官方定义
股票，是股份公司发行的所有权凭证。
公司为了筹集资金，把股份分给投资者，投资者买入后就成为股东。

02 说大白话
股票就是“你拥有一家公司的部分权益”的证明。

03 核心要点
买股票不是买一串代码，而是在买一家公司的部分所有权。
```

### 第 3 页｜公司为什么要发行股票？

```text
01 筹钱做发展
公司扩张、研发、建厂、开新业务，都需要资金。发行股票，本质上是一种融资方式。

02 不是借钱，而是引入股东
和贷款不同，发股票不用按期还本付息，但原股东持股会被稀释。

03 投资者换来什么
投资者出钱支持公司发展，未来如果公司经营得好，就有机会分享分红和股价上涨带来的收益。
```

### 第 4 页｜买股票，买到了什么？

```text
01 收益权
如果公司赚钱并愿意分红，股东可以获得股息；如果市场愿意给更高价格，股东也可能赚到差价。

02 股东权利
股票代表股东身份。你拥有知情、表决等股东权利，但普通投资者通常是很小的一份股权。

03 也要承担风险
公司经营变差、行业景气下降，或者市场情绪转弱，股价都可能下跌，投资者也可能亏损。
```

### 第 5 页｜股票为什么会涨跌？

```text
01 先看公司基本面
市场预期公司未来能赚更多钱，股价往往更容易上涨；如果业绩变差，股价往往承压。

02 再看供求和情绪
想买的人多、愿意出的价格更高，股价就容易涨；反过来，卖的人更多，股价就容易跌。

03 还受外部环境影响
利率、政策、行业景气度、突发事件，都会影响市场对公司的估值和信心。
```

### 第 6 页｜总结页

```text
这一期，你学会了吗？

1. 股票 = 公司所有权凭证
2. 买股票 = 成为股东，分享收益，也承担风险
3. 看懂股票，第一步不是盯K线，而是先理解公司

点赞收藏，和远山财经一起打好金融基础。
```

## 8. 生图提示词骨架

### 封面提示词骨架

```text
生成一张 3:4 竖版小红书财经知识封面图。
品牌为「远山财经」，必须使用正确 LOGO：山形图标 + 中文“远山财经” + 英文“YUANSHAN FINANCE”。
整体为米白纸感背景，极简、克制、知识感强。
主题色使用赭橙 / 陶土红 / 暖红棕，不使用绿色。
封面不放人物。
顶部小 LOGO，中上方放“股市入门｜第X期”标签，页面高度 1/3 到 2/5 区间放超大标题「{本期概念}」，标题下方放副标题「{一句问题}」。
底部只放极简装饰线和小型上涨折线图标。
不要页码，不要水印，不要复杂插图，不要下期预告。
```

### 内容页提示词骨架

```text
生成一张 3:4 竖版小红书财经知识内容页。
沿用「远山财经｜股市入门」系列模板：米白纸感背景，深棕黑文字，赭橙 / 陶土红强调色，不使用绿色。
顶部使用正确远山财经 LOGO。
可以加入固定人物 IP「远山学长」作为小老师辅助：年轻男性、圆框眼镜、白衬衫、蓝灰色开衫、卡其裤、温和老师感。
页面采用清晰的 3 个模块结构，每个模块包含编号、短标题、正文解释、简洁图标。
正文要大、清楚、适合小红书阅读。
不要堆砌文字，不要复杂装饰，不要页码，不要下期预告。
```
